<p align="center">Automated Mothership installer for DigitalOcean</p>

## Usage

* Clone repo
* `cd` into repo
* Run `yarn install`
* Run `yarn link`
* Run `mothership-setup`
* Provide your domain (without any subdomains) and DigitalOcean API token
